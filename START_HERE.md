# Quick Start

## 1. Setup Backend
Run the setup script once to create the virtual environment and install dependencies:
```powershell
.\setup_backend.ps1
```

## 2. Start Project
Open two terminals.

Terminal 1 (Backend):
```powershell
.\run_backend.ps1
```

Terminal 2 (Frontend):
```powershell
.\run_frontend.ps1
```
